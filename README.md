# 🎭 Emoji Mood Selector

An interactive webpage where users can select an emoji that reflects their current mood and get a personalized message in return.

---

## 🌟 Features
- Simple one-page layout  
- Clickable emoji mood options  
- Personalized message based on mood  
- Beginner-friendly code using only **HTML**, **CSS**, and **JavaScript**
